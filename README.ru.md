*   Описание:
*   Расширяет стандартную функцию random, добавляя поддержку seed.
*   
*   Использование:
*     - random(value) - обычная генерация
*     - random(value, .seed = значение) - генерация с указанным seed
*   
*   Дополнительные функции:
*   
*     SeedRandom(seed)
*       - Устанавливает начальное значение для генератора
*   
*     RandomEx(...)
*       - Принимает несколько значений и seed (последний аргумент)
*       - Возвращает сумму случайных значений для каждого аргумента
*   
*   Примеры:
*     [1] Обычное использование:
*         new rand = random(100); // 0-100
*   
*     [2] С указанием seed:
*         new rand = random(50, .seed = 12345);
*   
*     [3] Множественная генерация:
*         new sum = RandomEx(10, 20, 30, 12345);
*   
*   Алгоритм:
*     Линейный конгруэнтный генератор (LCG)
*     Xn+1 = (a * Xn + c) mod m
*     где:
*       a = 214013
*       c = 2531011
*       m = 2^31
*   
*   Зачем:
*     Стандартный генератор Random на сервере оказался предсказуем 
*     при известном seed. Этот модуль предоставляет альтернативный 
*     метод генерации случайных чисел.
